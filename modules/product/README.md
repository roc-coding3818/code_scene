# 商品模块

## 模块介绍

商品模块负责商品的管理，包括商品分类、SPU、SKU等核心概念。

## 功能清单

### 分类管理
- [ ] 分类树结构
- [ ] 分类增删改查
- [ ] 分类层级管理

### 商品管理
- [ ] 商品 SPU 管理
- [ ] 商品 SKU 管理
- [ ] 商品属性管理
- [ ] 商品上下架

### 商品搜索
- [ ] 关键词搜索
- [ ] 分类筛选
- [ ] 价格区间筛选
- [ ] 排序（价格/销量/新品）

## 核心知识点

- SPU 与 SKU 概念
- 商品属性设计
- ElasticSearch 商品搜索
- 商品缓存策略
